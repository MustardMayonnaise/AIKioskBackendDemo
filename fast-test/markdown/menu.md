맥도날드 테스트 메뉴 데이터이다.
아래 메뉴와 가격을 기준으로 주문을 계산한다.
가격 단위는 원이다.

## 1. 버거 단품

- 빅맥: 5,500
  - 구성: 소고기 패티 2장, 치즈, 양상추, 피클, 양파, 빅맥소스

- 더블치즈버거: 4,800
  - 구성: 소고기 패티 2장, 치즈 2장, 피클, 양파, 케첩, 머스타드

- 불고기버거: 3,800
  - 구성: 불고기 패티, 양상추, 불고기소스, 마요소스

- 맥스파이시 상하이버거: 5,600
  - 구성: 치킨 패티, 양상추, 토마토, 매운소스

- 베이컨토마토디럭스: 5,900
  - 구성: 소고기 패티, 베이컨, 토마토, 양상추, 치즈, 소스

- 슈슈버거: 5,000
  - 구성: 새우 패티, 양상추, 소스

## 2. 세트 구성

버거를 세트로 변경하면 기본적으로 아래가 포함된다.

- 후렌치 후라이 미디움
- 코카콜라 미디움

세트 추가 금액:
- 단품 가격 + 2,800

예:
- 빅맥 세트 = 5,500 + 2,800 = 8,300
- 불고기버거 세트 = 3,800 + 2,800 = 6,600

## 5. 세트 옵션 변경 규칙

아래 음료로 변경 가능하다.

- 코카콜라 제로 미디움: 추가금 없음
- 스프라이트 미디움: 추가금 없음
- 오렌지주스: +500
- 아이스 아메리카노: +800
- 생수: 추가금 없음

### 5-2. 사이드 변경
세트 기본 사이드는 후렌치 후라이 미디움이다.
아래 사이드로 변경 가능하다.

- 후렌치 후라이 라지: +500
- 치즈스틱 2조각: +400
- 맥너겟 4조각: +500
- 코울슬로: 추가금 없음

## 11. 없는 메뉴 처리 예시

아래 메뉴는 판매하지 않는다.

- 와퍼
- 징거버거
- 아이스크림
- 밀크쉐이크
- 치즈볼
- 샐러드

없는 메뉴를 요청하면 유사 메뉴를 제안한다.

예:
- 와퍼 → 빅맥 또는 더블치즈버거 제안
- 징거버거 → 맥스파이시 상하이버거 또는 불고기버거 제안
- 밀크쉐이크 → 코카콜라, 코카콜라 제로, 오렌지주스 제안

## 12. 가격 계산 규칙

총금액은 아래 기준으로 계산한다.

총금액 =
각 주문 항목의 기본 가격
+ 세트 추가 금액
+ 옵션 변경 추가 금액
+ 추가 토핑 금액
× 수량 합산

재료 제외는 가격 차감이 없다.

## 13. 주문 요약 예시

### 예시 1
- 빅맥 세트 1개
- 음료 코카콜라 제로
- 사이드 후렌치 후라이 라지

계산:
- 빅맥 5,500
- 세트 추가 2,800
- 코카콜라 제로 변경 0
- 후렌치 후라이 라지 변경 500
합계: 8,800

### 예시 2
- 불고기버거 단품 2개
- 치즈 추가 1회씩

계산:
- 불고기버거 3,800 x 2 = 7,600
- 치즈 추가 700 x 2 = 1,400
합계: 9,000

### 예시 3
- 더블치즈버거 세트 1개
- 사이드 치즈스틱 변경
- 음료 아이스 아메리카노 변경
- 피클 제외

계산:
- 더블치즈버거 4,800
- 세트 추가 2,800
- 치즈스틱 변경 400
- 아이스 아메리카노 변경 800
합계: 8,800

사이드는 후렌치 후라이, 치즈스틱, 맥너겟, 코울슬로가 있습니다.

음료는 코카콜라, 코카콜라 제로, 스프라이트, 오렌지주스, 아이스 아메리카노, 생수가 있습니다.
